# GNN Poverty Identification

A PyTorch and dgl implementation for the WWW 2023 paper below:
[Graph-based Village Poverty Identification](https://arxiv.org/pdf/2302.06862.pdf)

## Running
``tar -zxvf data.tar.gz ``
``python main.py ``
